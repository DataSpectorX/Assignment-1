Q1. Explain the difference between the following data types with examples.
Ans.Integer (int): Integer is used to store whole numbers. These numbers do not contain any decimal point.
Example: age = 18, marks = 95
Float (float): Float is used to store numbers that have decimal values.
Example: height = 5.8, price = 49.99
String (str): String is used to store text, words, or sentences. It is written inside single or double quotes.
Example: name = "Anurag", city = "Delhi"
Boolean (bool): Boolean can store only two values, True or False. It is mainly used for checking conditions.
Example: is_student = True, has_car = False
Q4. Explain any five commonly used string methods in Python with examples.
Ans.
upper(): This method converts all letters of a string into capital letters.
Example: "hello".upper() gives "HELLO"
lower(): This method converts all letters of a string into small letters.
Example: "PYTHON".lower() gives "python"
strip(): This method removes extra spaces from the beginning and end of a string.
Example: " Anurag ".strip() gives "Anurag"
replace(): This method replaces one word or character with another.
Example: "I like mango".replace("mango", "apple") gives "I like apple"
find(): This method finds the position of a character or word in a string.
Example: "computer".find("m") gives 2
If the character is not found, it returns -1.
Q7. What is Artificial Intelligence (AI)? Explain its importance and mention any four real-life applications of AI.
AnsArificial Intelligence (AI) is a technology that enables computers and machines to perform tasks that usually require human intelligence. AI can learn from data, solve problems, and make decisions.
Importance of AI:
AI is important because it helps people complete tasks faster and more accurately. It reduces human effort, saves time, and can process large amounts of data quickly. AI is being used in many fields to improve efficiency and productivity.
Four Real-Life Applications of AI:
Healthcare: AI helps doctors detect diseases and analyze medical reports.
Online Shopping: AI recommends products to customers based on their interests and previous purchases.
Navigation Apps: AI helps apps like Google Maps suggest the best routes and estimate travel time.
Education: AI-powered learning platforms provide personalized learning experiences for students.
